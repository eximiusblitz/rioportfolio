Correlation Between Stock Price and Its Commodity Price

This is a project created to answer a question, "Is there a correlation between a stock price (in this case, PGAS) and the commodity that the company sells (in this case, natural gas)?". To find the correlation, my friends and I processed the data first so there are no missing values or data mismatches, after that we normalize both dataset so that the price range are the same, and finally we calculate the correlation value using the Pearson method.
